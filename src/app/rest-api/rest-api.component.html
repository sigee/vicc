Nyilt REST Api hamarosan!

A response valtozhat még!

A válazs a szervertől JSON formátumban fog érkezni. Példa a JSON objektum válaszra: 

data: [

  _id: "58b243357f88ad21c49f4baa",
  cim: " Drog ",
  tartalom: " Unoka kérdezi: - Nagyi, nem láttad a tablettáimat? Az volt ráírva, hogy ... ",
  tag: "Bizarr",
  __v: 0

]

Jelenlegi REST Api url-ek:

  • Lapozás négyesével az adatbazisban:
    viccek.herokuapp.com/api/viccek/[oldal szám]
    Pl: viccek.herokuapp.com/api/viccek/0

  • Egy random vicc az egész adatbázisból:
    viccek.herokuapp.com/api/random

  • Viccek kategóriák szerint:
    viccek.herokuapp.com/api/viccek/[kategória]/[oldal szám]
    Pl: viccek.herokuapp.com/api/viccek/Bizarr/0

  • Keresés ID szerint:
    viccek.herokuapp.com/api/kereses/[ID]
    Pl: viccek.herokuapp.com/api/kereses/58b243357f88ad21c49f4baa
